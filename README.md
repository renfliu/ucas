## UCAS网络登录

命令行的方式登录UCAS网络

```
ucas.py is a python script for ucas network
usage: python[3] ucas.py command [params]
 	commands:
    	login [-p] [username] [password]
        	login to ucas network, e.g. python3 ucas.py login user123 pass123
            -p, --prefix: add '发\' before username
    	logout
        	logout from ucas network
    	find
        	find an account from ucas icc...
    	status
        	status of network
```
